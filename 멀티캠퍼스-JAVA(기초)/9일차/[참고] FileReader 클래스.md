# [참고] FileReader 클래스

## ✏️  FileReader

```java
FileReader fr = new FileReader(filename);
```

- 외부 파일을 읽어오기 위한 기능을 제공하는 클래스

## ✏️  API문서 살펴보기

![image](img/00-1.png)

- FileReader 클래스를 살펴보면 생성자의 호출 시에 예외를 발생시키도록 throw문이 작성되어 있는 것을 확인 할 수 있다
- 생성자를 호출함과 동시에 예외가 발생하므로, 컴파일할 수 없어 Compile-Exception에 해당한다
